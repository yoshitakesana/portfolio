食べられるか食べられないか

概要

このサイトは「食べられるか食べられないか」を判定するWebアプリです。 Flaskを使ったWebアプリケーションで、AIモデルを利用してアップロードされた画像が食べ物かどうかを判定します。 AIは汎用型モデルですが、ユーザーが「食べ物かどうか」と入力することで、食べ物専用の判定サイトとして動作する仕様になっています。 また、判定結果に応じて簡単な説明文も表示されるように設計されています。

技術スタック

  • Flask
  • HTML / CSS / JavaScript
  • AIモデル（Faster R-CNNなど物体検出モデル）
