思い出再現AIアプリ

製品概要

「過去の思い出の場所を“今の姿”で見ることはできても、当時の天気や雰囲気を再現するのは難しい」── そんな課題から生まれた、“思い出の風景をAIで再現する”サービスです。

座標（位置情報）と日付を入力するだけで、当日の天気データをもとにAIが風景画像を生成します。 写真がなくても、その時の空気感をもう一度感じることができます。

開発技術

  • フロントエンド: Next.js / React
  • バックエンド: Django REST Framework
  • 主要API: OpenAI API, Visual Crossing Weather API, Google Street View Static API
  • ライブラリ・モジュール: requests, Pillow (PIL), dotenv, openai, io, base64, os
  • デバイス: PCブラウザ（開発・検証用）

製品説明

本アプリは、地図座標（緯度・経度）と日付を入力すると、以下の3つのプロセスで動作します：

  1. Google Street Viewから現地画像を取得
  2. Visual Crossing Weather APIで当日の天気を取得
  3. OpenAIの画像生成モデルで天気に合わせた画像を生成

フロントエンド（Next.js）とバックエンド（Django REST Framework）が連携し、 シンプルで直感的に使えるUIを実現しました。
※精度が低いですが一番最後の写真がAI加工後の写真です。
